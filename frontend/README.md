### Frontend

- 프로젝트 생성 CRA - redux - typescript
  ```bash
  $ npx create-react-app frontend --template redux-typescript
  ```
  - 템플릿 이용 프로젝트 구성
- Axios, dotenv
  ```bash
  $ npm install axios dotenv
  ```
- React-router-dom
  ```bash
  $ npm install react-router-dom
  ```
- react-helmet-async
  ```bash
  $ npm install react-helmet-async
  ```
- socket.io-client
  ```bash
  $ npm install socket.io-client
  ```
